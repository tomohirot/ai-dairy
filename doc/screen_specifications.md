## 画面詳細

### S-001: ログイン画面

#### 画面概要
システムへのログイン認証を行う画面

#### 画面レイアウト
```
+----------------------------------+
|      営業日報システム            |
+----------------------------------+
|                                  |
|  メールアドレス: [____________]  |
|                                  |
|  パスワード:     [____________]  |
|                                  |
|         [ログイン]ボタン         |
|                                  |
+----------------------------------+
```

#### 入力項目

| 項目名 | 形式 | 必須 | バリデーション |
|--------|------|------|----------------|
| メールアドレス | テキスト | ○ | メール形式チェック |
| パスワード | パスワード | ○ | 8文字以上 |

#### アクション
- **ログインボタン**: 認証成功時、ダッシュボード(S-002)へ遷移

#### エラーメッセージ
- "メールアドレスまたはパスワードが正しくありません"

### S-002: ダッシュボード

#### 画面概要
自分または部下の日報一覧を表示し、未読コメントなどの通知を確認できる画面

#### 画面レイアウト
```
+------------------------------------------------------------------+
| [営業日報システム]                              [ログアウト]    |
+------------------------------------------------------------------+
| ナビゲーション: [ダッシュボード] [日報登録] [顧客マスタ] [営業マスタ※] |
+------------------------------------------------------------------+
|                                                                  |
| ようこそ、○○さん                                                |
|                                                                  |
| [新規日報作成]ボタン                                             |
|                                                                  |
| ■ 表示対象: [自分の日報 ▼]  期間: [今月 ▼]                     |
|                                                                  |
| +--------------------------------------------------------------+ |
| | 日報一覧                                                     | |
| +------+----------+--------+----------+--------+---------------+ |
| | 日付 | 営業名   | 訪問数 | Problem | Plan   | コメント     | |
| +------+----------+--------+----------+--------+---------------+ |
| | 12/10| 田中太郎 | 3件    | あり    | あり   | 新着1件 🔴  | |
| | 12/09| 田中太郎 | 2件    | あり    | あり   | 0件         | |
| | 12/08| 田中太郎 | 4件    | あり    | あり   | 確認済み    | |
| +------+----------+--------+----------+--------+---------------+ |
|                                                                  |
| ページネーション: < 1 2 3 ... 10 >                              |
+------------------------------------------------------------------+
```

#### 表示項目

| 項目名 | 説明 |
|--------|------|
| 日付 | 日報の報告日 |
| 営業名 | 日報作成者の名前 |
| 訪問数 | その日の訪問記録件数 |
| Problem | 課題・相談の有無 |
| Plan | 明日の計画の有無 |
| コメント | コメント件数と未読状態 |

#### フィルター機能

| フィルター | 選択肢 |
|-----------|--------|
| 表示対象 | 自分の日報 / 部下の日報 / 全ての日報（管理者のみ） |
| 期間 | 今日 / 今週 / 今月 / 先月 / カスタム期間 |

#### アクション
- **新規日報作成ボタン**: 日報登録画面(S-003)へ遷移
- **各行クリック**: 日報詳細画面(S-004)へ遷移
- **顧客マスタリンク**: 顧客マスタ一覧(S-005)へ遷移
- **営業マスタリンク**: 営業マスタ一覧(S-007)へ遷移（管理者のみ表示）

### S-003: 日報登録/編集画面

#### 画面概要
日報の新規作成または既存日報の編集を行う画面

#### 画面レイアウト
```
+------------------------------------------------------------------+
| [営業日報システム]                              [ログアウト]    |
+------------------------------------------------------------------+
| [← ダッシュボードに戻る]                                        |
+------------------------------------------------------------------+
|                                                                  |
| ■ 日報作成 / 編集                                               |
|                                                                  |
| 報告日: [2024/12/10 ▼]    営業名: 田中太郎                     |
|                                                                  |
| +--------------------------------------------------------------+ |
| | 訪問記録                                        [+ 追加]     | |
| +--------------------------------------------------------------+ |
| |                                                              | |
| | 訪問1:                                                       | |
| |   顧客: [株式会社サンプル ▼]  [新規顧客登録]                | |
| |   訪問内容:                                                  | |
| |   [________________________________________________]         | |
| |   [________________________________________________]         | |
| |                                              [× 削除]        | |
| +--------------------------------------------------------------+ |
| |                                                              | |
| | 訪問2:                                                       | |
| |   顧客: [選択してください ▼]                                 | |
| |   訪問内容:                                                  | |
| |   [________________________________________________]         | |
| |   [________________________________________________]         | |
| |                                              [× 削除]        | |
| +--------------------------------------------------------------+ |
|                                                                  |
| ■ 今日の課題・相談（Problem）                                   |
| +--------------------------------------------------------------+ |
| |                                                              | |
| |                                                              | |
| |                                                              | |
| +--------------------------------------------------------------+ |
|                                                                  |
| ■ 明日の予定（Plan）                                            |
| +--------------------------------------------------------------+ |
| |                                                              | |
| |                                                              | |
| |                                                              | |
| +--------------------------------------------------------------+ |
|                                                                  |
|                          [下書き保存] [登録/更新]               |
+------------------------------------------------------------------+
```

#### 入力項目

| 項目名 | 形式 | 必須 | バリデーション |
|--------|------|------|----------------|
| 報告日 | 日付選択 | ○ | 過去・今日のみ選択可 |
| 訪問記録 - 顧客 | ドロップダウン | ○ | 顧客マスタから選択 |
| 訪問記録 - 訪問内容 | テキストエリア | ○ | 1000文字以内 |
| 課題・相談 | テキストエリア | - | 2000文字以内 |
| 明日の予定 | テキストエリア | - | 2000文字以内 |

#### アクション
- **訪問記録追加ボタン**: 訪問記録入力欄を追加
- **訪問記録削除ボタン**: 該当の訪問記録を削除
- **新規顧客登録リンク**: 顧客マスタ登録画面(S-006)をモーダル表示
- **下書き保存ボタン**: データを保存（下書き状態）
- **登録/更新ボタン**: データを保存してダッシュボード(S-002)へ遷移

#### バリデーションルール
- 最低1件の訪問記録が必須
- 同一顧客への複数訪問は許可
- 訪問順序は入力順に自動採番

#### エラーメッセージ
- "訪問記録を最低1件は登録してください"
- "訪問内容を入力してください"
- "文字数制限を超えています"

### S-004: 日報詳細/閲覧画面

#### 画面概要
日報の詳細内容を表示し、上長がコメントを追加できる画面

#### 画面レイアウト
```
+------------------------------------------------------------------+
| [営業日報システム]                              [ログアウト]    |
+------------------------------------------------------------------+
| [← ダッシュボードに戻る]                       [編集]※          |
+------------------------------------------------------------------+
|                                                                  |
| ■ 日報詳細                                                      |
|                                                                  |
| 報告日: 2024年12月10日（火）                                    |
| 営業名: 田中太郎                                                 |
| 作成日時: 2024/12/10 18:30                                      |
|                                                                  |
| +--------------------------------------------------------------+ |
| | 訪問記録                                                     | |
| +--------------------------------------------------------------+ |
| | 【訪問1】株式会社サンプル                                    | |
| |   新商品のプレゼンテーションを実施。                         | |
| |   先方の関心は高く、次回見積提出の約束を取り付けた。         | |
| +--------------------------------------------------------------+ |
| | 【訪問2】株式会社テスト                                      | |
| |   既存契約の更新について打ち合わせ。                         | |
| |   価格面での調整が必要。                                     | |
| +--------------------------------------------------------------+ |
|                                                                  |
| ■ 今日の課題・相談（Problem）                                   |
| +--------------------------------------------------------------+ |
| | 株式会社テストの契約更新について、価格調整の権限を           | |
| | どこまで持つべきか相談したい。                               | |
| +--------------------------------------------------------------+ |
|                                                                  |
| ■ 明日の予定（Plan）                                            |
| +--------------------------------------------------------------+ |
| | ・株式会社サンプルへの見積書作成                             | |
| | ・株式会社テストへの価格調整案の提示                         | |
| | ・新規見込み客3社へのアポイント電話                          | |
| +--------------------------------------------------------------+ |
|                                                                  |
| ■ コメント                                                      |
| +--------------------------------------------------------------+ |
| | 山田部長 (2024/12/10 20:15)                                  | |
| | 株式会社テストの件、最大10%までの値引きは承認します。        | |
| | 明日の午前中に一度相談してください。                         | |
| +--------------------------------------------------------------+ |
|                                                                  |
| ※上長のみ表示                                                   |
| コメント追加:                                                   |
| +--------------------------------------------------------------+ |
| |                                                              | |
| |                                                              | |
| +--------------------------------------------------------------+ |
|                                               [コメント送信]    |
+------------------------------------------------------------------+
```

#### 表示項目
- 日報基本情報（報告日、営業名、作成日時）
- 訪問記録一覧（顧客名、訪問内容）
- 課題・相談（Problem）
- 明日の予定（Plan）
- コメント履歴

#### 入力項目（上長のみ）

| 項目名 | 形式 | 必須 | バリデーション |
|--------|------|------|----------------|
| コメント | テキストエリア | ○ | 1000文字以内 |

#### アクション
- **編集ボタン**: 日報登録/編集画面(S-003)へ遷移（本人のみ表示）
- **コメント送信ボタン**: コメントを保存して画面をリロード（上長のみ表示）

#### 権限
- 本人: 閲覧・編集可能
- 上長: 閲覧・コメント可能
- その他: 閲覧のみ（自部署の場合）

## 共通UI要素

### ヘッダー
全画面共通で表示

```
+------------------------------------------------------------------+
| [営業日報システム]                     [通知🔔] [ログアウト]    |
+------------------------------------------------------------------+
```

- システム名
- 通知アイコン（新着コメントがある場合に赤丸表示）
- ログアウトボタン

### ナビゲーション
ダッシュボード、日報関連画面で表示

```
+------------------------------------------------------------------+
| [ダッシュボード] [日報登録] [顧客マスタ] [営業マスタ]※         |
+------------------------------------------------------------------+
```

※管理者のみ表示

### 確認ダイアログ
削除操作時に表示

```
+----------------------------------+
|    削除の確認                    |
+----------------------------------+
|                                  |
| この操作は取り消せません。       |
| 本当に削除しますか？             |
|                                  |
```

## レスポンシブ対応

### PC表示（1024px以上）
- 上記レイアウトを基本とする
- サイドバーナビゲーション
- テーブル形式での一覧表示

### タブレット表示（768px〜1023px）
- ナビゲーションをハンバーガーメニュー化
- テーブルを2カラム表示に変更
- 入力フォームは1カラム

### スマートフォン表示（767px以下）
- ハンバーガーメニュー
- 全て1カラム表示
- テーブルはカード形式に変更
- 入力項目を縦積み

- フォントサイズ変更に対応
